## 시스템 개요

**dwisCOST**는 제조공장의 실적원가를 관리하는 통합 시스템입니다.

### 주요 기능
- ✅ **생산수불 관리**: Lot/Run 단위 생산실적 추적
- ✅ **원가계산**: 직접재료비, 간접재료비, 가공비 자동 집계
- ✅ **재공/재고 평가**: WIP 및 제품재고 금액 평가
- ✅ **자재관리**: 원자재, 부자재, 포장재 입출고 관리
- ✅ **매출관리**: 거래명세서 기반 매출 집계
- ✅ **결산증빙**: 월마감 증빙자료 자동 생성

### 기술 스택
- **Backend**: Spring Boot 3.4.0, MyBatis 3.0.3, MS SQL Server
- **Frontend**: Vue.js 3, RealGrid (데이터 그리드)
- **인증**: JWT, Spring Security, OAuth2

### 사업장 구분
- **본사 (HQ)**: 한국 생산 법인
- **VINA (VN)**: 베트남 생산 법인

### C0003001: 재료비집계

**경로:** `/c0003001`  
**권한:** 인증 필요

월별 재료비(직접재료비 + 간접재료비)를 집계합니다.

```sql
직접재료비 = SUM(투입금액) WHERE MAT_CLASS='원자재'

```

#### 주요 화면 구성
- 기준월 선택
- 사업장 선택 (본사/VINA)

### C0003002: 재료비배부

**경로:** `/c0003002`  
**권한:** 인증 필요

#### 기능 설명
집계된 재료비를 제품별로 배부합니다.

#### 배부 기준
- **BOM 기준 배부**: 자재 소요량 기준
- **면적 기준 배부**: 제품 면적 비율 기준
- **공통 배부**: 전체 생산량 비율 기준

#### 배부 공식
```
배부금액 = 투입금액 × 배부율
배부율 = 제품별사용량 / 전체사용량
```

### C0008002: 원가항목별 비용 (DOI_EXPEN_AMT)

**경로:** `/c0008002`  
**권한:** 인증 필요

#### 기능 설명
원가항목별로 비용을 집계합니다.

#### 원가항목 분류
- **MDAX**: 직접재료비 (원자재)
- **MIDX/MIAX**: 간접재료비 (부자재)
- **가공비**: 인건비, 감가상각비, 전력비 등

#### SQL 쿼리 ID
- `C0008002_Sch1`: 원가항목별 집계
- `C0008002_Sch2`: 원가항목 상세내역

#### 집계 로직
```sql
-- 제조경비 집계
SELECT EXPEN_SEL, EXPEN_SEL명, SUB_NAME, ITEM_NAME, SUM(ACCT_AMT)
FROM DOI_ACCT_EXPEN
WHERE EXPEN_SEL NOT LIKE 'M%'
GROUP BY EXPEN_SEL, EXPEN_SEL명, SUB_NAME, ITEM_NAME

UNION ALL

-- 재료비 집계
SELECT 
    CASE WHEN MAT_CLASS='원자재' THEN 'MDAX' ELSE 'MIAX' END AS EXPEN_SEL,
    CASE WHEN MAT_CLASS='원자재' THEN '직접재료비' ELSE '간접재료비' END AS EXPEN_SEL명,
    SUM(IN_AMT) AS 금액
FROM DOI_MATERIAL_RESC
GROUP BY MAT_CLASS
```

### C0008004: 제품별 투입 비용 (DOI_PROD_EXPN)

**경로:** `/c0008004`  
**권한:** 인증 필요

#### 기능 설명
제품별로 투입된 총원가와 단가를 계산합니다.

#### SQL 쿼리 ID
- `C0008004_Sch1`: 제품별 단가 계산
- `C0008004_Sch2`: 제품별 원가요소 상세

#### 단가 계산 로직
```sql
WITH base_cost AS (
    -- 가공비
    SELECT model, '가공비' c_cost,
           AVG(in_qty + OUT_QTY + loss_qty)/2 AS 환산수량,
           SUM([in]) AS 투입금액
    FROM doi_cost
    GROUP BY model
    
    UNION ALL
    
    -- 재료비
    SELECT 도우모델 AS model,
           CASE WHEN mat_class='원자재' THEN '직접재료비' ELSE '간접재료비' END,
           AVG(환산량) AS 환산수량,
           SUM(배부금액) AS 투입금액
    FROM DOI_MAT_COST
    GROUP BY 도우모델, mat_class
)
SELECT model,
       COALESCE(d_matcost / NULLIF(in_qty, 0), 0) AS dm_unitcost,      -- 직접재료비 단가
       COALESCE(i_matcost / NULLIF(chg_qty, 0), 0) AS di_unitcost,     -- 간접재료비 단가
       COALESCE(fab_cost / NULLIF(chg_qty, 0), 0) AS f_unitcost,       -- 가공비 단가
       dm_unitcost + di_unitcost + f_unitcost AS sum_unitcost          -- 총단가
FROM in_cost
```

#### 핵심 개념
- **투입수량 (in_qty)**: 실제 투입된 수량
- **환산수량 (chg_qty)**: `(투입수량 + 출고수량 + 손실수량) / 2` (평균재공)
- **직접재료비 단가**: 투입수량 기준
- **간접재료비/가공비 단가**: 환산수량 기준

#### 관련 테이블
- `DOI_COST` (재공평가)
- `DOI_MAT_COST` (재료비배부)
- `DOI_EXPEN_MATL` (비용자재)

SELECT

## 주요 비즈니스 프로세스

### 1. 월마감 프로세스

```
1. 생산실적 마감 (M0003000)
   └─> 공정별 실적 입력 및 확정

2. 자재투입 집계 (자동)
   └─> DOI_MATERIAL_RESC 생성

3. 생산수불 집계 (자동)
   └─> DOI_PROD_SUBUL, DOI_PROD_LOTRUN 생성

4. 재료비 배부 (C0003002)
   └─> DOI_MAT_COST 생성

5. 가공비 배부 (C0003005)
   └─> DOI_COST 생성 (가공비 부분)

6. 재공/재고 평가 (자동)
   └─> DOI_COST, DOI_STOCK_COST 업데이트

7. 단가 계산 (자동)
   └─> C0008004 조회 가능

8. 결산 증빙 조회 (C0008000)
   └─> 각종 집계표 확인 및 Excel 다운로드
```

### 2. 생산 실적 입력 흐름

```
FRONT 공정:
  PO 발행 → 적층 작업 (M0003005) → 검사 → Lot 생성 → Run Card 발행 (M0003007)

BACK#1 공정:
  Lot 투입 → 박리/에칭 (M0003001) → Run 생성 → QC → Rework (필요시, M0003008)

BACK#2 공정:
  Run 투입 → Cassette QC (M0003002) → Cell QC (M0003003) → 이물제거 → 포장 (M0003009)
```

### 3. 자재 관리 흐름

```
구매 발주 → 입고 (M0001001~M0001009) → 재고 확인 → 생산 투입 출고 → 
재고 차감 → 재고실사 (월말) → 재고조정
```

### 1. Lot/Run 추적 체계
- **Lot**: FRONT 공정에서 생성 (Panel 단위)
- **Run**: BACK 공정에서 생성 (Cassette 단위)
- 전 공정 이력 추적 가능

- **실제원가 (ACTUAL)**: 월 단위 실적 기반

- **가공비**: 환산수량 기준 배부

- 본사(HQ), VINA(VN) 분리 관리
