## 📋 작업 개요

### 목표
부서별 비용(DOI_DEPT_COST)을 다음 4개 테이블로 분할 배부:
1. **DOI_MFG_EXP**: 제조경비 (Manufacturing Expenses)
2. **DOI_SGA**: 판매관리비 (Sales, General & Administrative)
3. **DOI_RND**: 연구개발비 (Research & Development)
4. **DOI_CASET**: 카세트팀 비용 (Cassette Team Cost)

### 기준 월

```sql
ALTER TABLE DOI_DEPT ADD RND_YN VARCHAR(1) NULL;           -- 연구개발 여부
```

- **COST_DIST_RATE**:
  - 100.00: 25개 부서 (단일 배부)
  - 90.00: 3개 부서 (원가/총원가 혼합)

### 2. 검증 테이블 생성 (2025-11-23)

4개 테이블 모두 동일한 구조:

```sql
CREATE TABLE DOI_MFG_EXP (
    YYYYMM VARCHAR(6) NOT NULL,           -- 기준년월
    SEL_CODE VARCHAR(6) NOT NULL,         -- 선택코드
    SITE VARCHAR(4) NOT NULL,             -- 사이트
    FROM_DEPT_CODE VARCHAR(10) NOT NULL,  -- 부서코드
    FROM_DEPT_NAME NVARCHAR(100) NULL,    -- 부서명
    FROM_ACCT_CODE VARCHAR(10) NOT NULL,  -- 계정코드
    FROM_ACCT_NAME NVARCHAR(100) NULL,    -- 계정과목
    EXPEN_SEL NVARCHAR(8) NULL,           -- 경비구분
    EXPEN_SEL_NAME NVARCHAR(60) NULL,     -- 경비구분명
    ACCT_AMT BIGINT NULL,                 -- 차변금액
    PRIMARY KEY (YYYYMM, SEL_CODE, SITE, FROM_DEPT_CODE, FROM_ACCT_CODE)
);
```

#### 테이블별 설명
- **DOI_MFG_EXP**: 제조경비 검증 테이블
- **DOI_SGA**: 판매관리비 검증 테이블
- **DOI_RND**: 연구개발비 검증 테이블
- **DOI_CASET**: 카세트팀 비용 검증 테이블

## 💡 해결 방안

### 옵션 1: 복수 테이블 배부

**COST_DIST_RATE의 의미를 "해당 카테고리로 가는 비율"로 해석**

#### 제조경비 (DOI_MFG_EXP)
```sql
-- 1) 원가 100% 부서 → 100% 제조경비
WHERE d.COST_DIST = N'원가' 
  AND d.DEPT NOT IN ('400','448')
  AND ISNULL(d.COST_DIST_RATE, 100) = 100

-- 2) 원가 90% 부서 → 90%만 제조경비 (나머지 10%는 판관비로)
WHERE d.COST_DIST = N'원가' 
  AND d.COST_DIST_RATE = 90

-- 3) 총원가 10% 부서 → 나머지 90%를 제조경비로
WHERE d.COST_DIST = N'총원가'
  AND d.RND_YN = 'N'
  AND d.COST_DIST_RATE = 10
  -- 금액: dc.차변금액 * 0.9
```

#### 판매관리비 (DOI_SGA)
```sql
-- 1) 총원가 100% 부서 → 100% 판관비
WHERE d.COST_DIST = N'총원가'
  AND d.RND_YN = 'N'
  AND ISNULL(d.COST_DIST_RATE, 100) = 100

-- 2) 총원가 10% 부서 → 10%만 판관비
WHERE d.COST_DIST = N'총원가'
  AND d.RND_YN = 'N'
  AND d.COST_DIST_RATE = 10
  -- 금액: dc.차변금액 * 0.1

-- 3) 원가 90% 부서 → 나머지 10%를 판관비로
WHERE d.COST_DIST = N'원가'
  AND d.COST_DIST_RATE = 90
  -- 금액: dc.차변금액 * 0.1
```

#### 연구개발비 (DOI_RND)
```sql
-- RND_YN='Y' 부서 → 100% 전액 배부 (현행 유지)
WHERE d.RND_YN = 'Y'
  -- 금액: dc.차변금액 (COST_DIST_RATE 무시)
```

#### 카세트팀 (DOI_CASET)
```sql
-- DEPT='400' 또는 '448' → UTG 비율 배부 (현행 유지)
WHERE d.DEPT IN ('400','448')
  -- 금액: dc.차변금액 * r.UTG
```
