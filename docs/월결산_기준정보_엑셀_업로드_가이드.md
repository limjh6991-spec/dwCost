# 월 결산 기준정보 — 엑셀 가공 · 업로드 가이드 (예: 2026-06 HQ)

원가/손익 결산에 앞서, ERP 등에서 내려받은 **엑셀 파일을 가공하여 기준정보 테이블에 업로드**하는 절차입니다.
- 대상 DB: **도우제조원가시스템** (10.100.40.17) — 운영 앱(10.100.40.19)이 참조
- 공통 키: `YYYYMM='202606'`, `SEL_CODE='ACTUAL'`, `SITE='HQ'` (월/사업장에 맞게 변경)
- 대부분 각 테이블의 **엑셀 업로드 화면**에서 해당 시트를 가공(헤더/TOTAL 제거·필터) 후 붙여넣기/업로드. **DOI_ACCT·DOI_MODEL_MAST는 예외**(아래 6·7절).

## 6. 수동 ① DOI_ACCT (전월 복사)

원가계정정보는 엑셀이 아니라 **전월 데이터를 복사해 월만 바꿔** 넣습니다.

```sql
-- 예: 202605 → 202606 복사 (필요 컬럼은 실제 스키마에 맞춰 조정)
INSERT INTO DOI_ACCT (YYYYMM, SEL_CODE, SITE, /* ...나머지 컬럼... */)
SELECT '202606', SEL_CODE, SITE, /* ...나머지 컬럼... */
FROM DOI_ACCT
WHERE YYYYMM='202605' AND SITE='HQ';
```
- 복사 후 신규/변경 계정, VN 총원가_순서 등 있으면 보정. (적재 확인: 202606 HQ 211건)

## 7. 수동 ② DOI_MODEL_MAST (쿼리로 업데이트)

`dw_product_mast`의 규격(Spec)을 `X × Y`로 분해하여 **면적(XY)** 을 만들어 적재합니다.
(면적 XY는 경비 배부기준 `dist_in`의 가중치로 쓰이므로 **누락 없이** 넣어야 합니다.)

### 7-0. 선행 작업 — `dw_product_mast` 이관 (필수)

아래 INSERT 쿼리는 **도우제조원가시스템**의 `dw_product_mast`를 원천으로 읽습니다. 이 데이터는 **MES 운영 DB(`도우제조MES시스템DB`)의 `dw_product_mast` 뷰**가 최신이므로, **쿼리 실행 전에 MES운영DB → 원가시스템으로 데이터를 새로 옮겨야** 합니다.

- **순서**: ① 원가시스템의 기존 `dw_product_mast` 데이터 **전량 삭제** → ② MES운영DB `dw_product_mast` 뷰 데이터를 원가시스템으로 이관
- ⚠️ 이관을 건너뛰면 **이전 월/구버전 규격**으로 면적(XY)이 계산되어 배부가 틀어집니다.

```sql
-- (원가시스템에서 실행) 1) 기존 데이터 삭제
DELETE FROM dw_product_mast;   -- 또는 TRUNCATE TABLE dw_product_mast;

-- 2) MES운영DB 뷰 → 원가시스템으로 이관
--    (동일 서버/링크드서버면 4-part 경로, 아니면 SSMS 데이터 내보내기/가져오기로 이관)
INSERT INTO dw_product_mast   -- (컬럼은 실제 스키마에 맞춰 지정)
SELECT * FROM [도우제조MES시스템DB].dbo.dw_product_mast;
```

> 서버가 분리되어 있으면 링크드서버 대신 **SSMS → 도우제조MES시스템DB → dw_product_mast 뷰 결과를 조회 → 도우제조원가시스템 dw_product_mast로 가져오기(Import/Export 마법사)** 방식으로 옮깁니다.

### 7-1. DOI_MODEL_MAST 적재 쿼리

이관 완료 후 아래 쿼리를 실행합니다.

```sql
With Spec_split1 as (
    select Model,upper(Spec) spec,Inch,Glass_thick,Sheet,Block,Cell,RUN_SIZE,
        substring(Spec,1,PATINDEX('%[*|×|X|x]%',Spec)-1) x,
        substring(Spec,PATINDEX('%[*|×|X|x]%',Spec)+1,len(Spec)) y
    from dw_product_mast
    where PATINDEX('%[*|×|X|x]%',Spec)> 0
), Spec_split2 as (
    select model, spec,Inch,Glass_thick,Sheet,Block,Cell,RUN_SIZE,
        replace(case when PATINDEX('%[±|M]%',x)=0 then x else substring(x,1,PATINDEX('%[±|M]%',x)-1) end,',','.') x,
        replace(case when PATINDEX('%[±|M]%',y)=0 then y else substring(y,1,PATINDEX('%[±|M]%',y)-1) end,',','.') y,
        row_number() over (partition by model order by x desc, y desc) as rn
    from Spec_split1
), Spec_split3 as (
    select '202606' yyyymm,'ACTUAL' sel_code,'HQ' site,model,
        spec,Inch,Glass_thick,Sheet,Block,Cell,RUN_SIZE,
        x,
        replace(case when PATINDEX('%*%',y)=0 then y else substring(y,1,PATINDEX('%*%',y)-1) end,',','.') y
    from Spec_split2
    where rn=1
)
insert into doi_model_mast
(YYYYMM,SEL_CODE,SITE,MODEL,SPEC,INCH,GLASS_THICK,SHEET,BLOCK,CELL,RUN_SIZE,X,Y,XY)
select *,cast(x as real)* cast(y as real) xy
from Spec_split3
order by model;
```
- 월(`'202606'`)을 대상 월로 바꿔 실행. (적재 확인: 202606 HQ 331건)
- ⚠️ **면적(XY) 누락 모델**이 있으면 경비 배부(`dist_in`)가 0/왜곡되므로, 결측 모델은 별도 보정.
