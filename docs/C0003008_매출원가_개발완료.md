# C0003008 매출원가 화면 개발 완료

## 📋 개요
**메뉴**: C0003000 제조매출원가 > C0003008 매출원가  
**목적**: 모델별/거래선별 매출금액 비율로 재고금액 배부하여 매출원가 계산  
**개발일**: 2025-11-11

---

## 🎯 핵심 로직

### 매출원가 계산식
```
매출원가 = 출고금액(OUT_AMT) × (거래처별 매출금액 / 모델별 총 매출금액)
```

### 배부율 계산식
```
배부율(%) = (거래처별 매출금액 / 모델별 총 매출금액) × 100
```

### 2. Backend (SQL Query)
- **파일**: `src/main/resources/mapper/web/c0003000/C0003000.xml`
- **Query ID**: `C0003008_Sch1`

#### SQL 로직 구조
```sql
WITH stco AS (
    -- 재고금액 데이터 (DOI_STOCK_COST)
    -- 모델별/구분별 출고수량, 출고금액 집계
),
sale AS (
    -- 매출 데이터 (DOI_SALE_RESC)
    -- 거래처별/품명별 매출수량, 매출금액 집계
),
sale_total AS (
    -- 모델별 총 매출금액 (배부 기준)
)
SELECT
    거래처,
    매출금액,
    출고금액,
    배부율 = (매출금액 / 모델별총매출금액) * 100,
    매출원가 = 출고금액 * (매출금액 / 모델별총매출금액)
FROM sale JOIN stco JOIN sale_total
```

#### 주요 테이블
1. **DOI_STOCK_COST**: 재고금액 (출고금액 기준)
2. **DOI_SALE_RESC**: 매출 정보 (거래명세서)

## 📊 데이터 흐름

```
1. 매출 데이터 수집 (DOI_SALE_RESC)
   └─> 거래처별/품명별 매출금액 집계

2. 재고 데이터 수집 (DOI_STOCK_COST)
   └─> 모델별/구분별 출고금액 집계

3. 배부율 계산
   └─> (거래처 매출금액 / 모델별 총매출금액) × 100

4. 매출원가 계산
   └─> 출고금액 × 배부율 / 100

5. 결과 표시
   └─> 거래처별 매출원가 + 원가항목별 상세
```

## ✅ 검증 포인트

### 1. 배부율 합계 검증
```sql
-- 같은 모델/구분에 대한 배부율 합계는 100% 이어야 함
SELECT 
    MODEL, 구분, 
    SUM(배부율) as 배부율합계
FROM C0003008_결과
GROUP BY MODEL, 구분
-- 배부율합계 = 100.00% (오차 ±0.01% 허용)
```

### 2. 매출원가 합계 검증
```sql
-- 같은 모델/구분/원가항목의 매출원가 합계는 출고금액과 일치해야 함
SELECT 
    MODEL, 구분, EXPEN_SEL,
    SUM(매출원가) as 매출원가합계,
    MAX(출고금액) as 출고금액
FROM C0003008_결과
GROUP BY MODEL, 구분, EXPEN_SEL
-- 매출원가합계 ≈ 출고금액 (반올림 오차 허용)
```
