### Step 3: 파라미터 설정 (4~6줄)

```sql
DECLARE @START_YYYYMM CHAR(6) = '202508';  -- 시작 월
DECLARE @END_YYYYMM CHAR(6) = '202510';    -- 종료 월  
DECLARE @SITE VARCHAR(10) = 'HQ';          -- 사이트 (HQ/VN/ALL)
```

### Step 4: 실행

- **F5** 키 또는 **실행 버튼** 클릭

### Step 5: 결과 확인

5개의 결과 탭이 표시됩니다:

1. **생산수불 통계** - 총건수, 음수재고, 수식오류
2. **재고수불 통계** - 총건수, 음수재고, 수식오류
3. **생산→재고 연결** - 누락, 불일치 건수
4. **음수 재고 상세** - 즉시 수정이 필요한 항목
5. **종합 요약** - 전체 현황 및 다음 단계 안내

### Step 6: 결과 저장

각 탭에서:
```
우클릭 → 결과를 다른 이름으로 저장 → Excel 형식
파일명: 진단결과_202511.xlsx
```
