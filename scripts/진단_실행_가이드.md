1. **SSMS 실행 및 연결**
   ```
   서버: 172.16.200.204
   데이터베이스: 도우제조MES시스템TEST
   사용자: TEST_MES_USER
   비밀번호: Dowoo1!
   ```

2. **진단 SQL 파일 열기**
   - 파일 메뉴 → 열기 → 파일
   - `src/sql/Validation_CurrentStatus_Analysis.sql` 선택

3. **파라미터 수정** (파일 상단 15~17줄)
   ```sql
   DECLARE @START_YYYYMM CHAR(6) = '202508';  -- 시작 월 변경
   DECLARE @END_YYYYMM CHAR(6) = '202510';    -- 종료 월 변경
   DECLARE @SITE VARCHAR(10) = 'HQ';          -- 사이트 변경 (HQ/VN/ALL)
   ```

4. **실행**
   - F5 키 또는 실행 버튼 클릭
   - 결과가 여러 탭으로 표시됨

5. **결과 확인 및 저장**
   - 각 결과 그리드에서 마우스 우클릭
   - "결과를 다른 이름으로 저장" → Excel 형식 선택
   - 파일명: `진단결과_202511.xlsx`

### 옵션 3: 간단 진단 쿼리 (즉시 실행)

빠르게 현황만 파악하려면 아래 쿼리를 복사해서 실행하세요:

```sql
-- ========================================
-- 빠른 진단 쿼리 (5분 완성)
-- ========================================

-- 1. 생산수불 기본 통계
SELECT 
    '1. 생산수불 통계' AS 구분,
    YYYYMM,
    DW_SITE,
    COUNT(*) as 총건수,
    SUM(CASE WHEN EOH_MONTH < 0 THEN 1 ELSE 0 END) as 음수재고건수,
    SUM(CASE WHEN ABS(EOH_MONTH - (BOH_MONTH + IN_MONTH + ISNULL(BONUS_MONTH,0) - OUT_MONTH - ISNULL(LOSS_MONTH,0) - ISNULL(NG_MONTH,0))) > 0.01 THEN 1 ELSE 0 END) as 수식오류건수
FROM DOI_PROD_SUBUL
WHERE YYYYMM BETWEEN '202508' AND '202510'  -- 기간 변경
    AND DW_SITE = 'HQ'  -- 사이트 변경 (HQ/VN)
GROUP BY YYYYMM, DW_SITE
ORDER BY YYYYMM, DW_SITE;

-- 2. 재고수불 기본 통계
SELECT 
    '2. 재고수불 통계' AS 구분,
    YYYYMM,
    SITE,
    COUNT(*) as 총건수,
    SUM(CASE WHEN EOH < 0 THEN 1 ELSE 0 END) as 음수재고건수,
    SUM(CASE WHEN ABS(EOH - (BOH + INPUT_PROD + INPUT_PURCH - OUT_INVOICE - OUT_ADJ)) > 0.01 THEN 1 ELSE 0 END) as 수식오류건수
FROM DOI_STOCK
WHERE YYYYMM BETWEEN '202508' AND '202510'  -- 기간 변경
    AND SITE = 'HQ'  -- 사이트 변경 (HQ/VN)
GROUP BY YYYYMM, SITE
ORDER BY YYYYMM, SITE;

-- 3. 생산→재고 연결 검증 (핵심)
SELECT 
    '3. 생산→재고 연결' AS 구분,
    P.YYYYMM,
    COUNT(*) as 총건수,
    SUM(CASE WHEN S.INPUT_PROD IS NULL THEN 1 ELSE 0 END) as 재고누락건수,
    SUM(CASE WHEN ABS(P.OUT_MONTH - ISNULL(S.INPUT_PROD, 0)) > 0.01 THEN 1 ELSE 0 END) as 수량불일치건수
FROM DOI_PROD_SUBUL P
LEFT JOIN DOI_STOCK S ON P.YYYYMM = S.YYYYMM 
    AND P.DW_SITE = S.SITE 
    AND P.도우모델 = S.MODEL
WHERE P.YYYYMM BETWEEN '202508' AND '202510'  -- 기간 변경
    AND P.DW_SITE = 'HQ'  -- 사이트 변경
    AND P.구분 IN ('제품', '반제품')
GROUP BY P.YYYYMM
ORDER BY P.YYYYMM;

-- 4. 종합 요약
SELECT 
    '종합 요약' AS 구분,
    (SELECT COUNT(*) FROM DOI_PROD_SUBUL WHERE YYYYMM BETWEEN '202508' AND '202510' AND DW_SITE = 'HQ' AND EOH_MONTH < 0) as 생산음수재고,
    (SELECT COUNT(*) FROM DOI_STOCK WHERE YYYYMM BETWEEN '202508' AND '202510' AND SITE = 'HQ' AND EOH < 0) as 재고음수재고,
    '→ 즉시 수정 필요' as 조치사항;
```

## 📊 결과 해석

### ✅ 정상 기준
- 음수재고건수: **0건**
- 수식오류건수: **0건**
- 재고누락건수: **0건**
- 수량불일치건수: **0건**

### ⚠️ 주의 필요
- 음수재고건수: **1~5건** → 우선 수정
- 수식오류건수: **1~10건** → 데이터 재계산
- 수량불일치건수: **1~10건** → 생산-재고 매핑 확인

### 🔴 긴급 조치
- 음수재고건수: **5건 이상** → 물리 실사 필요
- 수식오류건수: **10건 이상** → 시스템 로직 점검
- 수량불일치건수: **10건 이상** → I/F 타이밍 점검
