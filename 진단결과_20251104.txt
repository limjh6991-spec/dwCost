1. 8136 (202509) - 차이: 16,254 (EOH: 23,734, 계산값: 7,480)
2. 8136 (202508) - 차이: 12,582 (EOH: 37,399, 계산값: 24,817)
3. 902K (202508) - 차이:  7,147 (EOH:  3,940, 계산값: -3,207)
4. 8122 (202509) - 차이:  5,952 (EOH:    407, 계산값: -5,545)
5. 902K (202509) - 차이:  5,312 (EOH: 18,951, 계산값: 13,639)

⚠️ 주요 패턴:
- 모델 8136: 2개월 연속 큰 차이 발생
- 모델 902K: 2개월 연속 차이 발생
- 계산값이 음수인 경우 다수 (데이터 입력 오류 추정)

  ├─ OUT_ETC (기타출고)
  └─ OUT_MOVING (이동출고)
- EOH (기말재고)

📝 주의: INPUT_PURCH 컬럼 없음 (구매입고는 INPUT_ETC에 포함 추정)

📁 상세 데이터 확인 방법
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

SSMS에서 실행:

-- 오류 상세 조회 (모델 8136)
SELECT * 
FROM DOI_PROD_SUBUL 
WHERE 도우모델 = '8136' 
  AND YYYYMM IN ('202508', '202509')
ORDER BY YYYYMM;

-- 원천 데이터 확인
SELECT * 
FROM DW_PROD_FRONT 
WHERE MODEL = '8136' 
  AND YYYYMM IN ('202508', '202509');

-- 재고 연결 확인
SELECT P.*, S.INPUT_PROD
FROM DOI_PROD_SUBUL P
LEFT JOIN DOI_STOCK S 
  ON P.YYYYMM = S.YYYYMM 
  AND P.DW_SITE = S.SITE 
  AND P.도우모델 = S.MODEL
WHERE P.도우모델 = '8136';
