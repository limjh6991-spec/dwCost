## 🔧 해결한 주요 문제들

### 문제 1: 한글 컬럼명 오류
**증상**: `열 이름 'DW_MODEL'이(가) 유효하지 않습니다`

**원인**: SQL Server에서 한글 컬럼명을 직접 사용

**해결**:
```sql
SELECT 
    [도우모델] AS DW_MODEL,
    [model] AS MODEL,
    [Inch] AS INCH
FROM DOI_PROD_SUBUL
```

---

### 문제 2: RealGrid 데이터 미표시
**증상**: 데이터 조회는 성공(69건)하지만 그리드에 표시 안 됨

**원인**: 
- axios 응답 구조가 직접 배열 형태 (`Array(69)`)
- RealGrid의 데이터 바인딩 방식 불일치

**해결**: C0007003 스타일의 단순화된 방식 적용
```javascript
let param = {
  menuId: 'c0007006',
  queryId: 'selectBalanceCheck',
  queryParams: params,
  target: this.dataGridRows,  // target 방식
};
await this.$axios.api.search(param);
```

### 문제 3: Vue 개발 서버 포트 문제
**증상**: CORS 오류, 포트 4301로 시작됨

**해결**: 포트 4300으로 재시작

### 문제 4: 탭과 RealGrid 호환성 ⭐ **가장 까다로웠던 문제**

#### 증상
- 탭 추가 후 그리드가 **컬럼 헤더조차 표시 안 됨**
- 데이터는 조회됨 (69건)
- 엑셀 다운로드는 정상 작동
- 탭 없이는 정상 작동

#### 시도한 해결 방법들 (실패)

1. **v-if → v-show 변경**
   ```vue
   <!-- 실패: 여전히 표시 안 됨 -->
   <div v-show="activeTab === 0">
     <RealGrid ref="dataGrid" ... />
   </div>
   ```

2. **activeTab watch + resetSize() 호출**
   ```javascript
   // 실패: resetSize() 호출해도 해결 안 됨
   this.$watch('activeTab', (newTab) => {
     if (newTab === 0) {
       this.$nextTick(() => {
         const gridView = this.$refs.dataGrid.getGridView();
         if (gridView) {
           gridView.resetSize();
         }
       });
     }
   });
   ```

3. **RealGrid 문서 검색**
   - display:none 상태에서 그리드 크기가 0이 되는 문제 확인
   - resetSize() 권장되었으나 작동하지 않음

#### 최종 해결 방법 ✅

**참조 화면 분석**: TAB014000 (사용자메뉴 권한관리)

**핵심 발견**:
- 탭 내부에 **직접 그리드 배치** (v-show로 감싸지 않음)
- **명시적인 높이 지정** (`height: 500px`)
- Bootstrap Vue의 탭 메커니즘이 자동으로 표시/숨김 처리

**적용한 코드**:
```vue
<div class="grid_box">
  <b-tabs v-model="activeTab" class="custom-tabs">
    <b-tab title="생산 수량 밸런스 체크">
      <div class="left_box">
        <div class="btn_wrap ms-auto">
          <b-button class="second" @click="excelBtnClick">엑셀 다운로드</b-button>
        </div>
      </div>
      <!-- ✅ 핵심: height 명시 + v-show 없이 직접 배치 -->
      <div class="grid-border-none" style="height: 500px;">
        <RealGrid ref="dataGrid" :uid="'dataGrid'" :step="'1'" :rows="dataGridRows" style="height: 100%" />
      </div>
    </b-tab>
    <b-tab title="일자별 연속성 체크">
      <div class="grid-border-none" style="height: 500px;">
        <div style="padding: 40px; text-align: center; color: #6c757d;">
          일자별 연속성 체크 기능은 준비 중입니다.
        </div>
      </div>
    </b-tab>
  </b-tabs>
</div>
```

**왜 이 방식이 작동했는가?**:
1. **b-tab 컴포넌트**가 내부적으로 탭 전환 처리
2. **v-show 제거**로 display:none 문제 회피
3. **명시적 높이 지정**으로 그리드가 크기 계산 가능
4. 그리드가 **활성 탭에서만 초기화**되어 정상 렌더링
